Adding an Image to CursorCraft Homepage

The code has been updated to include an image in the homepage. Follow these steps to add your own image:

Option 1: Use a placeholder service temporarily

Until you have your own image, you can use a placeholder image service:

  1. Download an 800x450 placeholder image from placehold.co
  2. Save it as "hero-image.jpg" in the "public/images" folder of your project

Option 2: Use your own image

  1. Prepare an image with a 16:9 aspect ratio (ideally 800x450 pixels or larger)
  2. Name it "hero-image.jpg" (or update the code to match your filename)
  3. Place it in the "public/images" folder of your project

Note: The image component is set up with:

  • fill property to fill its container
  • object-cover class to maintain aspect ratio while covering the area
  • Container has aspect-video (16:9 ratio) and rounded corners

After adding your image, run your development server with npm run dev to see the changes.
